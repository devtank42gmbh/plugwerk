About this plugin

ACME PDF Export provides high-fidelity PDF generation for any PF4J-based application. Built on Apache PDFBox with a fluent builder API, it allows you to export any view, report, or data set as a professionally formatted PDF document.

Features

  • Template-based PDF generation (Mustache / FreeMarker)
  • Custom fonts, headers, and footers
  • Digital watermarks and security settings
  • Batch processing with progress callbacks
  • Password protection and permission flags
  • Table of contents generation

Quick Start

// Get the service from the plugin manager
PdfExportService pdfService = pluginManager
    .getExtensions(PdfExportService.class)
    .get(0);

// Export a model as PDF
byte[] pdf = pdfService.builder()
    .template("report.html")
    .model(reportData)
    .watermark("CONFIDENTIAL")
    .build()
    .export();
